# Project Roadmap

## Phase 1: Dual-Brain Stability (Current)
- [x] UART communication protocol.
- [x] Basic pH/Temp control on Brain A.
- [x] Dashboard and MQTT on Brain B.

## Phase 2: Advanced Sensing
- [ ] Integration of AS7341 spectral sensor for NADH monitoring.
- [ ] Dissolved Oxygen (DO) probe integration (I2C or Galvanic).
- [ ] CO2 off-gas analysis.

## Phase 3: Intelligence & Automation
- [ ] Automated pH calibration using standard buffer detection.
- [ ] Predictive growth modeling (calculating µ in real-time with better filtering).
- [ ] Fed-batch automation based on OD/pH triggers.

## Phase 4: Production Hardware
- [ ] Custom PCB for dual ESP32-S3.
- [ ] Integrated pump drivers (TMC2208 for silent operation).
- [ ] Waterproof enclosure with industrial connectors.
